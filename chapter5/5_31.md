#### 词典格式的转换
------
```perl
#程序功能：将词表数据重新格式化并输出
#实现思想：用正则表达式提取dict.txt中每行单词和译文，
#并放入哈希变量中重新输出

open(In,"dict.txt");
while(<In>){
	chomp;
	#用正则表达式提取单词和译文	
	if(/(.+)=>(.+)/){
		#哈希变量赋值：以词条中的单词作为哈希变量的关键字，以词条中的译文
		#作为哈希变量的值
		$hash{$1}=$2;
	}
}
close(In);

#遍历哈希变量
foreach(sort keys %hash){
	print "$_\t$hash{$_}\n";
}

```
